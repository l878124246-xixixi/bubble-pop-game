# 🫧 戳泡泡舒压游戏

一个简单有趣的舒压小游戏，1分钟就能玩！

## 🎮 游戏特色

- 🫧 真实的泡泡图片，带有彩虹光泽
- 💥 戳破时的爆裂动画和彩虹水滴飞溅效果
- 🔊 清脆的音效反馈
- 📱 支持手机和电脑
- 🎯 25个泡泡，轻松完成

## 🚀 在线体验

点击这里玩：[开始游戏](https://wb_liushaopeng.github.io/bubble-pop-game/)

## 💻 本地运行

1. 下载这个项目
2. 直接打开 `index.html` 文件即可

## 📝 说明

这是一个纯前端项目，使用HTML + CSS + JavaScript制作，无需任何服务器或依赖。

享受戳泡泡的乐趣吧！✨
